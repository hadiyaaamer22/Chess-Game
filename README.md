# Chess-Game
OOP Semester Project
This is a simple console based chess game in C++.

## Features
- Two player chess game
- Piece movement rules
- Board display
- Turn switching
- King capture ends the game

## Files
- Piece.h contains all chess piece classes
- Board.h contains board class
- Board.cpp contains board functions
- main.cpp contains the game loop

## How to Run
Compile main.cpp and Board.cpp together.
